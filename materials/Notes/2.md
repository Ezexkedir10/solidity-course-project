Address Interactions
Sending Ether

Ether is the cryptocurrency of Ethereum.

Sending Ether Example
payable(msg.sender).transfer(1 ether);
Important Concepts
Concept	Description
payable	allows address to receive Ether
transfer	sends Ether
msg.sender	current user address
Message Calls

Contracts can communicate with other contracts using message calls.

Example
contract A {
    function hello() public pure returns(string memory) {
        return "Hello";
    }
}

Another contract can call this function.

Reverting Transactions

Transactions can fail and revert changes.

Methods
Method	Purpose
require()	validates conditions
revert()	manually revert
assert()	internal error checking
Example
require(balance > 0, "Insufficient balance");
Calling Contracts

One smart contract can interact with another.

Example
contract B {
    function callHello(address contractAddress) public {
        A(contractAddress).hello();
    }
}
