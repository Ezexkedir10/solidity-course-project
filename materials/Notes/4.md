Voting Contract

Voting contracts allow users to vote securely on blockchain.

Features
Transparent voting
Immutable records
Prevent duplicate voting
Example
mapping(address => bool) public voted;
Inheritance

Inheritance allows contracts to reuse code from other contracts.

Example
contract Animal {
    function sound() public pure returns(string memory) {
        return "Animal Sound";
    }
}

contract Dog is Animal {
}

Dog inherits functions from Animal.

Benefits of Inheritance
Code reuse
Easier maintenance
Cleaner architecture
