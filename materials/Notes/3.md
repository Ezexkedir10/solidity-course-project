Reference Types Notes
Arrays

Arrays store multiple values of the same type.

Example
uint[] public numbers;
Adding Values
numbers.push(10);
Structs

Structs group multiple variables together.

Example
struct Student {
    string name;
    uint age;
}
Creating Struct
Student public s1 = Student("John", 21);
Mappings

Mappings store key-value pairs.

Example
mapping(address => uint) public balances;
Usage
balances[msg.sender] = 100;

Mappings are commonly used for:

balances
ownership
voting systems
