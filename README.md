# solidity-course-project

Group project for the "Introduction to Blockchain" laboratory (Solidity course — Alchemy University).

Section 3
Computer Science and Engineering
Introduction to Blockchain laboratory assignment

## Brief Project Description
This repository contains the Solidity practice exercises and laboratory assignments completed by the group. Each student maintains a subfolder with Solidity source files (`code/`) and corresponding screenshots (`screenshot/`). The examples cover basic types, functions, sending/receiving Ether, arrays, mappings, structs, inheritance and voting contracts.
